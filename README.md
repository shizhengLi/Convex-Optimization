# 凸优化知识点与实战应用

这个项目全面记录了凸优化的核心知识点和实际应用，特别专注于金融行业中的优化包使用实战。包含理论讲解、数学公式、Python可视化示例、实际投资组合优化案例，以及CVXOPT源代码深度解析。

## 核心内容

### 📚 基础理论
1. **凸集合和凸函数**
   - 凸集合的定义和直观理解
   - 凸函数的定义、判断方法和性质
   - 常见的凸集合和凸函数示例
   - Jensen不等式及其可视化

2. **凸优化问题**
   - 凸优化问题的标准形式
   - 线性规划、二次规划、半定规划
   - 对偶理论和KKT条件
   - 最优性条件和算法基础

### 🛠️ 专业优化包详解
1. **MOSEK - 锥优化专家**
   - 专业的凸优化求解器
   - 在二阶锥优化(SOCP)和半定优化(SDP)方面表现优异
   - 金融优化应用：投资组合、风险管理、衍生品定价
   - 实际案例：风险平价、鲁棒优化、参数化求解

2. **Gurobi - 混合整数规划领导者**
   - 业界最快的优化求解器之一
   - 在MIP问题上性能卓越
   - 运筹学和供应链管理应用
   - 实际案例：指数跟踪、交易策略优化、多目标优化

3. **CVXOPT - 开源学术工具**
   - 完全免费的开源优化包
   - 与Python科学计算生态深度集成
   - 适合学术研究和教学
   - 实际案例：L1正则化、稀疏投资组合、有效前沿计算
   - **新增**：CVXOPT源代码深度解析，包含算法实现细节和性能优化技巧

### 💼 金融行业实战应用
1. **投资组合优化**
   - 马科维茨均值-方差模型
   - 考虑交易成本的再平衡策略
   - 风险预算投资组合
   - 约束条件：最小/最大权重、换手率限制

2. **风险管理**
   - CVaR最小化
   - 鲁棒优化处理参数不确定性
   - 压力测试和情景分析

3. **实际案例演示**
   - 6类资产的投资组合优化
   - 夏普比率从0.78提升到0.81
   - 求解时间仅0.006秒
   - 完整的有效前沿分析
   - **新增**：基于CVXOPT源码分析的优化技术实战应用

### 📊 可视化示例
- **凸集合 vs 非凸集合**：直观展示凸集合的性质
- **凸函数示例**：常见凸函数的图形对比
- **Jensen不等式**：凸函数核心性质可视化
- **线性规划**：资源分配问题的求解过程
- **二次规划**：等高线和约束条件可视化
- **投资组合优化**：有效前沿和权重分布
- **优化包对比**：性能和质量对比分析
